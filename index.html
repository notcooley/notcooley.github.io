Debugging FPGAs Through a Software Slowdown

When there are many chips onboard a satellite, and many systems on each chip; debugging a single component is nearly impossible. I created a test suite for a Spacewire router on an FPGA allowing for custom data packets, scenarios, and logs for better debugging. All packaged into a user friendly shell.

PlantPico

This was a toy project for a Christmas gift I made. It is a moisture sensor with a UI to evaluate plants' moisture, and if it was ready to be watered. It was good fun, and my first experience with coding microcontrollers in Python rather than C. It was also a good opportunity to use more components like LCDs, buttons, and sensors.

Mormir Basic

Mormir Basic is a format of MTG that typically can only be played online. Take out a majority of the skill, let luck take the wheel. You get random creatures from the thousands in Magic's history, and make do. Now, with the help of a few buttons and a reciept printer, you can play it in real life with your friends!
